# BMW Smoke Tests

Быстрый smoke-набор встроен в `bmw.html`.

## Запуск

1. Открой `bmw.html`.
2. В DevTools Console выполни:

```js
runBmwSmokeTests({ notify: true })
```

## Автозапуск

Можно открыть страницу с параметром `?smoke=1`, например:

`http://localhost:8080/bmw.html?smoke=1`

## Что проверяется

1. Наличие публичного модульного API (`AppModules`).
2. Переключение вкладки на `Объекты`.
3. Сохранение и восстановление фильтров через `localStorage`.
4. Работа чекбокса `Только в работе`.
5. Работа чекбокса `Показать выполненные`.
6. Мгновенное применение цвета инспектора к видимым точкам.
7. Отсутствие лишнего ререндера секции списков объектов при неизменных данных.
